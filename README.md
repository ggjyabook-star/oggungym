# Oggun Gym - La Forja de la Fuerza

Sitio web completo e interactivo para **Oggun Gym**, diseñado con un enfoque estético premium (tema verde neón/esmeralda sobre modo oscuro profundo) y construido con tecnologías web nativas (HTML5, CSS3, JavaScript).

## Características del Proyecto
- **Página de Inicio (`index.html`)**: Landing page moderna con filosofía, zonas, planes de membresía, calculadora de IMC interactiva y formulario de contacto. Incluye un video looping de fondo en la sección de filosofía.
- **Catálogo de la Forja (`catalogo.html`)**: Buscador interactivo y filtros en tiempo real para reservar clases de élite, adquirir suplementación avanzada y consultar la ficha técnica del equipamiento del gimnasio.
- **Carrito de Reservas**: Carrito lateral interactivo que almacena la información de manera persistente (`localStorage`) y genera un mensaje estructurado de checkout listo para enviar por WhatsApp a recepción.
- **Panel de Usuario Premium (`dashboard-usuario.html`)**: Zona privada con selector de rutinas de fuerza (incluida rutina de Espalda detallada por el Head Coach), guía de uso de maquinaria, macros nutricionales personalizados para déficit calórico y registro histórico de peso y plicometría.
- **Panel de Administración (`dashboard-admin.html`)**: Consola del staff con métricas generales de facturación, barra de búsqueda de socios, toggles de estado de pago interactivos y botones inteligentes de WhatsApp para contacto y cobros directos.
- **Identidad Corporativa**: Integración del logotipo oficial de **Oggun Team** y enlaces a las redes oficiales (Facebook).
